# Crypto Dashboard

A lightweight, responsive cryptocurrency dashboard built with React and Tailwind CSS. Streams live market data (price, market cap, 24h change) for the top coins from the CoinGecko API, refreshing every 60 seconds, with instant client-side search. Falls back to bundled sample data when offline or rate-limited.

## Stack

- **React 19** + **Vite**
- **Tailwind CSS 4**
- **CoinGecko API** (no key required)

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Build

```bash
npm run build
npm run preview
```
